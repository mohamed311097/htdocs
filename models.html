Good Practice & UI

methods of good practice: The definition of MGP is the guideline design layout that's required to provide the best & up to date website possible. Now in addition to sustainind up to date designs but also the intent in which you want you website to be fundementally based upon. For example if you required your website to be founded upon usability (i.e navigation), with the MGP guideline you could attain all the necessary requirement to help you provide the best possible website while achieving all of the necessary content and information you wish to add. Also the main factor of MGP is it provides you with the guidlines to attain minimilism, like for example by providing 3-6 options instead of 10-15 it allows you to make it easier for users to fully navigate.

There are so many different current website that have mastered the art of showing up to date website that provide revolutionary content layout making it easier for users to access certain and specific information with ease. All the major companies like (Apple, Samsung, Nike, Sony) have all mastered this art.The usage of banners have also become a huge element in the web design world; why?

Well as we all know images can tell a thousand words, and with that in mind newer sites have become more image eccentric , by attaching more meaningful images to the website in ways that provides information but on a more vibrant and easy way, providing users with not a lot of reading to do, if they want to find something out. In conclusion, with the information that i have provided i have followed the MGP as much as possible to try and attain an easy and simple website for users to navigate through .
