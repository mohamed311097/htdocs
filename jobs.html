JOB DESCRIPTION

Web Developer

Job Summary

Web developers build and maintain websites and web applications. Although their work usually focuses solely on the underlying software and databases (known as the 'back end'), some web developers work on the interface and visual design (the 'front end'), while others combine both ('full-stack development').

writing code in one or more programming or scripting languages, such as PHP or JavaScript, plus also planning and prototyping new applications in some jobs designing the architecture of the components of an application is part of the job and finally also testing new features thoroughly to ensure they perform the correct task in all cases. In simplified terms learning and testing new technologies, and languages and if you're an open minded indivual and eager, then this is for you.

Salary: £25,000 yearly

Game Designer

Job Summary

Working as a game designer you'll be involved in the creation and production of games for personal computers, games consoles, social/online games, arcade games, tablets, mobiles and other devices. Your work will usually be concerned with either design (including art and animation) or programming.

When it comes to the day to day job role their are certain requirements necessary of each designer being capable of achieving. For example developing designs or initial concept designs for games including game play. Finally using 2D or 3D modelling and animation software, such as Maya, at the production stage. These different elements are the normal standards required to become a professional game designer and if you're an open minded indivual and eager, then this is for you.

Salary: £25,000 yearly

Programmer

Job Summary

Computer programmers write code to create software programs. They turn the program designs created by software developers and engineersinto instructions that a computer can follow. They are required to write programs in a variety of computer languages, such as C++, Python, Ruby and Java. Also updating and expanding existing programs are also requierd .

Furthermore your role would also require the debugging of programs by testing for and fixing errors. These are the initial elements required to become a fully fledged programmer. Some programs are relatively simple and usually take a few days to write. Other programs, like computer operating systems, are more complex and can take a year or more to complete.

Salary: £31,000 yearly
